# x86 emulator

## How to run?

```
// for ocaml implementation (deprecated)
dune exec x86emulator -- --dbg-mode --machine i386 ./test/helloworld_org.bin

// for rust implementation
cargo run -- --dbg-mode --machine i386 ./test/helloworld_org.bin

```
